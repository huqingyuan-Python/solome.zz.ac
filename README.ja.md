# 🌐 パーソナルポートフォリオ

> パーソナルポートフォリオ — パーソナルワークとプロジェクトを展示する静的ウェブサイト。シンプルで优雅なデザインで、ダーク/ライトテーマ切り替え対応。

**Language / 语言 / 言語:** [简体中文](./README.md) · [English](./README.en.md) · [日本語](./README.ja.md) · [繁體中文](./README.zh-TW.md)

## ✨ 機能特徴

- 🎨 **ミニマリストデザイン** — シンプルで优雅なビジュアル、コンテンツ本身を強調
- 🌙 **ダーク/ライトテーマ** — ワンク릭で切り替え、パーソナライズされたブラウジング体験
- 📱 **レスポンシブレイアウト** — 各种の画面サイズに完美対応
- 🌐 **多言語サポート** — 简体中文、繁体中文、英語、日本語
- ⚡ **高速読み込み** — ピュア静的ページ、サーバーなしでアクセス可能

## 🚀 クイックスタート

### オンラインアクセス

https://solome.zz.ac にアクセスして表示。

### ローカル実行

```bash
git clone https://github.com/huqingyuan-Python/solome.zz.ac.git
cd solome.zz.ac
# ブラウザで en.html または他のページファイルを直接開く
```

## 📖 使い方

1. ページを開き、ダークまたはライトテーマを選択
2. パーソナルワークとプロジェクトの展示を閲覧
3. 関連リンクをクリックして詳細を確認
