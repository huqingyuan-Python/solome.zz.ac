# 🌐 个人作品集

> Personal Portfolio — 一个展示个人作品与项目的静态网站页面，简洁优雅的设计风格，支持深色/浅色主题切换。

**Language / 语言 / 言語:** [简体中文](./README.md) · [English](./README.en.md) · [日本語](./README.ja.md) · [繁體中文](./README.zh-TW.md)

## ✨ 功能特性

- 🎨 **简约设计风格** — 简洁优雅的视觉效果，突出内容本身
- 🌙 **深色/浅色主题** — 一键切换，打造个性化的浏览体验
- 📱 **响应式布局** — 完美适配各种屏幕尺寸
- 🌐 **多语言支持** — 简体中文、繁体中文、英语、日语
- ⚡ **快速加载** — 纯静态页面，无需服务器即可访问

## 🚀 快速开始

### 在线访问

直接访问 [https://solome.zz.ac](https://solome.zz.ac) 即可查看。

### 本地运行

```bash
git clone https://github.com/huqingyuan-Python/solome.zz.ac.git
cd solome.zz.ac
# 直接用浏览器打开 en.html 或其他页面文件
```

## 📖 使用说明

1. 打开页面，选择**深色**或**浅色**主题
2. 浏览个人作品与项目展示
3. 点击相关链接了解更多详情

## 🔧 技术栈

| 技术 | 说明 |
