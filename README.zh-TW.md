# 🌐 個人作品集

> Personal Portfolio — 一個展示個人作品與項目的靜態網站頁面，簡潔優雅的設計風格，支援深色/淺色主題切換。

**Language / 语言 / 言語:** [简体中文](./README.md) · [English](./README.en.md) · [日本語](./README.ja.md) · [繁體中文](./README.zh-TW.md)

## ✨ 功能特點

- 🎨 **簡約設計風格** — 簡潔優雅的視覺效果，突出內容本身
- 🌙 **深色/淺色主題** — 一鍵切換，打造個性化的流覽體驗
- 📱 **響應式佈局** — 完美適配各種螢幕尺寸
- 🌐 **多語言支援** — 簡體中文、繁體中文、英語、日語
- ⚡ **快速載入** — 純靜態頁面，無需伺服器即可訪問

## 🚀 快速開始

### 在線訪問

直接訪問 [https://solome.zz.ac](https://solome.zz.ac) 即可查看。

### 本地運行

```bash
git clone https://github.com/huqingyuan-Python/solome.zz.ac.git
cd solome.zz.ac
# 直接用流覽器打開 en.html 或其他頁面檔
```

## 📖 使用說明

1. 開啟頁面，選擇**深色**或**淺色**主題
2. 流覽個人作品與項目展示
3. 點擊相關連結了解更多詳情
