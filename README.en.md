## ✨ Features

- 🎨 **Minimalist Design** — Clean and elegant visual style that highlights content
- 🌙 **Dark/Light Theme** — One-click toggle for personalized browsing experience
- 📱 **Responsive Layout** — Perfectly adapts to various screen sizes
- 🌐 **Multi-language Support** — Simplified Chinese, Traditional Chinese, English, Japanese
- ⚡ **Fast Loading** — Pure static pages, accessible without a server

## 🚀 Quick Start

### Online Access

Visit [https://solome.zz.ac](https://solome.zz.ac) to view the site.

### Local Run

```bash
git clone https://github.com/huqingyuan-Python/solome.zz.ac.git
cd solome.zz.ac
# Open en.html or other page files directly in your browser
```

## 📖 Usage

1. Open the page and choose dark or light theme
2. Browse personal works and project showcases
3. Click relevant links for more details
